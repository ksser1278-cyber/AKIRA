## 핵심 작곡 패턴

### 곡 구조 (일반적)
```
Intro → A-Melo → B-Melo → Sabi (Chorus) → A-Melo' → B-Melo' → Sabi' → C-Melo (Bridge) → Last Sabi → Outro
```

### Ado 특유의 패턴
1. **다이나믹 대비**: 조용한 A멜로 → 폭발적 사비 (거의 모든 곡)
2. **보컬 전환점**: B멜로에서 사비로 넘어가는 순간 보컬 모드 체인지
3. **후반 에스컬레이션**: 라스트 사비에서 한 옥타브 올리거나 더 격렬하게
4. **감정 아크**: 억제 → 해방 → 폭발의 3단계 감정 곡선
