# YOASOBI (Ayase) 가사 심층 분석 — 63곡 코퍼스 기반

> 분석 대상: `lyrics/raw/yoasobi/` 내 63곡 (12곡 일본어 원문 정독 + 영어 번역 크로스체크)

---

## 1. Ado와의 근본적 차이: YOASOBI는 분석이 의미 있다

Ado와 달리 **YOASOBI의 모든 곡은 Ayase 한 명이 작사/작곡**한다. 프로듀서별 편차가 없기 때문에 일관된 "Ayase 문체"가 존재하며, 이를 분석하고 재현하는 것이 가능하다.

또한 YOASOBI의 사운드는 **신스팝/일렉트로닉 팝** 기반으로, SUNO가 상대적으로 잘 재현하는 장르에 해당한다.

---

## 2. Ayase 가사의 핵심 특성

### 2.1 소설 원작 → 서사 구조

YOASOBI의 정체성 자체가 **"소설을 음악으로"**이다. 거의 모든 곡이 특정 단편소설을 원작으로 한다.

| 곡 | 원작 소설 |
|----|---------|
| 夜に駆ける | 타나토스의 유혹 (星野舞夜) |
| 群青 | 或る画家の手記 (一実) |
| 怪物 | 自分の胸に自分の耳を押し当てて |
| 祝福 | 기동전사 건담 수성의 마녀 |
| アイドル | 오시노코 원작 |

→ 가사가 **스토리의 특정 장면/감정을 압축**한 구조. "감정을 노래한다"보다 **"이야기의 한 순간을 포착한다"**는 것이 Ado와의 최대 차이점.

---

### 2.2 Ayase의 문체 DNA

**A. 「」 대화 삽입 — 가장 뚜렷한 시그니처**

거의 모든 곡에 「」로 감싼 직접화법이 등장한다.

```
「さよなら」だけだった                 ← 夜に駆ける: 핵심 트리거
「好きだよ」                          ← あの夢をなぞって: 첫 줄+마지막 줄
「おかえり」思わず零れた言葉は 違うな   ← たぶん: 감정 반전의 축
「終わりにしたい」だなんてさ            ← 夜に駆ける: 아이러니 트위스트
「キミにきめた!」                      ← ポケモン曲: 선언
「はじめまして」した日から              ← ポケモン曲: 시작점 마커
```

→ **원칙: 「」 안의 단어가 곡 전체의 감정 피벗 포인트. 가사의 "핵"이자 리프레인의 앵커.**

**B. 구어체 내성 독백 — 회화적 사고 흐름**

Ayase의 가사는 "노래"보다 **"혼잣말"**에 가깝다. 문장이 끝나지 않고 이어지고, 되묻고, 다시 말을 바꾼다.

```
悪いのは誰だ 分かんないよ 誰のせいでもない たぶん    ← たぶん
いや、幸せだった 確かにほら 救われたんだよ あなたに  ← 怪物(JP)
そんなもんさ これでいい                              ← 群青
それでもきっといつかはきっと僕らはきっと              ← 夜に駆ける
```

→ 특징: `たぶん`, `きっと`, `ほら`, `さ`, `んだ`, `かな` 같은 **구어 종결어미**가 압도적으로 많다. 문어체가 거의 없다.

**C. 핵심어 반복 + 미세 변형 (Morphological Shift)**

같은 프레이즈를 반복하되 **조사나 활용형을 한 글자만** 바꿔서 감정 방향을 전환한다.

```
会いたくて 会いたくて → 会いたくて 会えなくて        ← ~たくて(희망) → ~なくて(불가)
騒がしい日々に笑えない君に → 騒がしい日々に笑えなくなっていた ← 현재→과거 전환
知らず知らず隠してた → 知らず知らず隠してた (반복)     ← 완전 반복=강조
僕の手を掴んでほら → 繋いだ手を離さないでよ          ← 능동→수동=관계 변화
何枚でも ほら何枚でも → 何回でも ほら何回でも        ← 枚(장)→回(번) 단위 교체
もうちょっと どうか変わらないで → もうちょっと どうか終わらないで ← 変わる→終わる 동사 교체
```

→ **원칙: 한 음절/한 글자의 변화가 감정 벡터를 뒤집는 것이 Ayase 가사의 핵심 기법.**

**D. 이미지 클러스터 — 반복되는 상징 체계**

63곡에서 반복적으로 등장하는 이미지군:

| 이미지 축 | 구체적 키워드 | 의미 |
|-----------|-------------|------|
| **밤↔아침** | 夜, 明けない夜, 朝, 日が昇る, 夜明け | 불안↔해방 |
| **빛↔어둠** | 光, ヒカリ, 暗闇, 影, 眩しい | 희망↔절망 |
| **색** | 青い, 透明, 色とりどり, 金色 | 자아의 상태 |
| **시간** | チックタック, 過ぎる日々, 未来, あの日 | 현재↔과거 |
| **달리기** | 駆ける, 走り続け, 歩いていく, 飛べる | 전진/도주 |
| **손** | 手を掴んで, 繋いだ手, 手を振る, 差し伸べた手 | 연결/이별 |
| **별/하늘** | 星, 空, 流れ星, 花火 | 꿈/순간 |

→ 이 이미지들은 곡마다 조합이 달라지지만, **"밤에서 아침으로 가는 여정"**이라는 큰 틀 안에서 움직인다.

**E. 감정 아크: 자기 의심 → 불안 속 전진 → 불완전한 긍정**

Ado가 "억제→폭발→해방"이라면, YOASOBI는 **"불안→망설임→그래도 한 걸음"**이다.

```
[Verse] 불안/자기 의심
  「怖いことだけど」「自信なんかない」「分かんないよ」

[Pre-Chorus] 망설임 속 결심의 순간
  「それでも」「だから」「きっと」

[Chorus] 불완전하지만 전진
  「大丈夫」「行こう」「あとは楽しむだけだ」「走りだす」

[Bridge] 취약함의 고백
  「本当は僕も言いたいんだ」

[Final Chorus] 첫 Chorus보다 한 발짝 더 나간 긍정
  「もう今はあの日の透明な僕じゃない」
```

→ **Ado의 "0→100 폭발"과 달리, YOASOBI는 "30→45→60→75"의 점진적 상승.** SUNO에서 이걸 구현하려면 섹션별 에너지를 세분화해야 한다.

### 2.3 오노마토페아/음감어 — 감각 리듬

YOASOBI도 오노마토페를 적극 활용하지만, Ado와 용법이 다르다:

```
チックタック        ← 시계 소리 (시간 경과)
ドキドキ            ← 두근거림 (설렘/긴장)
ビリビリ            ← 찌릿찌릿 (전투적 흥분)
ジリジリ            ← 조여오는 (긴장 고조)
ヒリヒリ            ← 따끔따끔 (고통)
ギリギリ            ← 아슬아슬 (한계)
キラキラ            ← 반짝반짝 (희망)
ぎゅっと            ← 꽉 (밀착)
くしゃくしゃ        ← 구겨진 (웃는 얼굴)
```

→ Ado: 오노마토페 = 폭발/타격감 (`ガタガタ`, `ギャーギャー`)
→ YOASOBI: 오노마토페 = 감각/감정의 테크스쳐 (`ドキドキ`, `キラキラ`)

## 3. SUNO 프롬프트 설계 시사점

### YOASOBI가 SUNO에 적합한 이유

1. **일렉트로닉 팝 기반** — SUNO가 잘 재현하는 장르
2. **ikura의 보컬 스타일** — 밝고 클리어, 극단적 기교 없음 → SUNO 보컬로 근사치 가능
3. **점진적 빌드업** — 급격한 전환 없이 자연스러운 곡 구조 → SUNO 안정성 확보
4. **영어 혼용 패턴** — SUNO가 일영 혼합을 비교적 잘 처리

### Style 태그 설계 방향

```
Tempo 130 BPM, J-Pop, Synth-Pop, Electronic Pop, Bright Female Vocal,
Piano-Driven, Layered Synth Arpeggios, Emotional Build-up,
Catchy Melodic Hook, Clean Vocal, Close-Mic Intimate Verses,
Wide Chorus Release, Uplifting
```
